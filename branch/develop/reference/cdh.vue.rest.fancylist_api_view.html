Defines a filter for a field.

Options for a filter are automatically generated by Vue

Field should be given as a string, in the format {attribute}(.{subattribute})*

(read, just like you would access an attribute of an object in Python, but sans the object itself and as a string).

For example: “subObject.created”

Parameters:

  • field ({str}) – the field that is sortable

  • display_name ({str}) – Text that should be displayed on the button

class SortDefinition(field: str, label: str, asc: bool = True, desc: bool = True)[source]¶

Bases: object

Defines a sort option for a field

Field should be given as a string, in the format {attribute}(.{subattribute})*

(read, just like you would access an attribute of an object in Python, but sans the object itself and as a string).

For example: “subObject.title”

Parameters:

  • field ({str}) – the field that is sortable

  • label ({str}) – The text to display in the dropdown. Direction arrows are added automatically

  • default ({"asc"|"desc"|None}) –

    If not None, this direction will be the sort applied upon load. If multiple defs are set as default, the first one encountered will be used

  • asc ({bool}) – whether sorting ascendingly is supported

  • desc ({bool}) – whether sorting descendingly is supported
