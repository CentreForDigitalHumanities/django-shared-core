class cdh.vue3.components.uu_list.serializers.DDVLinkField(*args, **kwargs)[source]¶

Bases: Field

DDVLinkField is a custom DRF serializer, which will format its output for use in DDVLink, DDVButton and DDVAction DDV columns.

When using this field for DDVLink or DDVButton, you can use this class as the serializer. DDVAction columns should use the DDVActionsField serializer instead.

Note: classes is ignored when using DDVLink; use the column classes argument instead
